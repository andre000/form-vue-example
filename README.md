## Commands

### Install

```sh
npm install
```

### Build

```sh
npm run build
```

### Usage

```sh
npm run serve
```

### Run unit tests

```sh
npm test:unit
```

### Run end-to-end tests

```sh
npm test:e2e
```
or

```sh
npm test:e2e -- --headless
```

### Lints and fixes files
```sh
npm run lint
```

### Run Storybook
```sh
npm run storybook:serve
```

- [vee-validate](https://logaretm.github.io/vee-validate/): Used for form validation
